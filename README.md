# Encriptador de Texto – Alura LATAM

Este proyecto es parte del Challenge #1 del programa Oracle ONE en Alura LATAM.  
Consiste en una aplicación web que permite encriptar y desencriptar mensajes de texto utilizando reglas específicas de sustitución de vocales.  
Además, ofrece la funcionalidad de copiar el resultado al portapapeles.

## 🔐 Reglas de Encriptación

Cada vocal en el texto se reemplaza según las siguientes claves:

- `a` → `ai`
- `e` → `enter`
- `i` → `imes`
- `o` → `ober`
- `u` → `ufat`

Por ejemplo:

- Texto original: `gato`
- Texto encriptado: `gaitober`

## ✅ Requisitos del Proyecto

- Solo se permiten letras minúsculas.
- No se admiten letras con acentos ni caracteres especiales.
- Debe ser posible encriptar y desencriptar el texto ingresado.
- La interfaz debe permitir al usuario elegir entre encriptar o desencriptar.
- El resultado debe mostrarse en pantalla.
- Incluir un botón que copie el texto resultante al portapapeles.

## 🛠️ Tecnologías Utilizadas

- HTML
- CSS

## 🚀 Cómo Ejecutar el Proyecto

1. Clona el repositorio:

   ```bash
   git clone https://github.com/Itemt/Encriptador-AluraLATAM.git
   ```

2. Navega al directorio del proyecto:

   ```bash
   cd Encriptador-AluraLATAM
   ```

3. Abre el archivo `index.html` en tu navegador preferido para utilizar la aplicación.
